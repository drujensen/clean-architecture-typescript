# Frontend Application

This is a React application built with TypeScript and Vite, using Tailwind CSS for styling.

## Features

- Product CRUD operations (Create, Read, Update, Delete)
- Responsive design using Tailwind CSS
- Minimal dependencies following React best practices

## Structure

- `src/App.tsx` - Main application component with product management
- `src/main.tsx` - Entry point
- `src/index.css` - Tailwind CSS imports

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
